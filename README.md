# MR Cloth Store (Web Development Project)

## Description
MR Cloth Store is a web development project built using HTML, CSS, JS, PHP, and MySQL.
It demonstrates a basic e-commerce-style backend with database connectivity.

## Features
- Product listing
- Backend API using PHP
- Database connection
- Image handling

## Technologies Used
- HTML, CSS, JS
- PHP
- MySQL

## How to Run Locally
1. Install XAMPP / WAMP
2. Place project in `htdocs`
3. Import `db_setup.sql` into MySQL
4. Open `index.html` in browser

## Note
This project is for learning and portfolio purposes.
